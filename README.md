# sformat HDL

[![Analytics](https://ga-beacon.appspot.com/UA-68153177-5/README.md?pixel)](https://github.com/suoto/sformat_hdl)

sformat HDL is based on Easics' [PCK_FIO][pck_fio] and aims to provide C-like
string formatting.
